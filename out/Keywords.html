(static) keywordSetup(keyword, enableDoubleOptIn, confirmationMessage, joinMessage, forwardToEmail, forwardToUrl, groups, cb) → {any}

Configures a rented keyword for use.
Parameters:
Name Type Description
keyword String Required. Must be rented.
enableDoubleOptIn Bool Optional. True, and the keyword will support double-opt. False, and it won't.
confirmationMessage String Optional. A message a contact will receive upon texting the keyword. (e.g. "Reply Y to join our sweetest list.")
joinMessage String – Optional. A message a contact will receive upon accepting the confirmation message (e.g. "Thank you for joining our sweetest list.")
forwardToEmail String Optional. Must be a valid email address. Inbound messages (MO) will be forwarded to the specified email address.
forwardToUrl String – Optional. Must be a valid url. Inbound messages (MO) will be forwarded to the specified url.
groups Array Optional. An array of valid group ids. Contacts will be auto-joined into the specified group(s).
